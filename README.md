# Text Mining

This repository contains lab exercises and coursework for a Text Mining course. It covers various natural language processing (NLP) techniques and text analysis methods through 6 hands-on labs.

## Contents

- **[Lab1](Lab1/)** - Introduction to Text Mining basics
- **[Lab2](Lab2/)** - Text preprocessing and analysis
- **[Lab3](Lab3/)** - Advanced text processing techniques
- **[Lab4](Lab4/)** - Text classification and information extraction
- **[Lab5](Lab5/)** - Topic modeling and text representation
- **[Lab6](Lab6/)** - Text classification and representation learning

## Technologies Used

- Python
- Jupyter Notebooks
- Natural Language Processing (NLP) libraries
- Machine Learning for text analysis

## Structure

Each lab folder contains:
- Jupyter notebooks (`.ipynb`) with code implementations
- Lab reports (`.pdf`) documenting the work
- Data files where applicable
